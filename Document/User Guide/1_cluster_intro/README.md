## 1.1.  클러스터 구축 목적

첫번째, 다수의 사용자가 다수의 시스템을 사용하기 위해  
운영체제 설치, 응용프로그램 및 라이브러리 설치, 사용자 계정 동기화 등의  
사용환경을 구축하는 작업을 간소화 하기위해 클러스터로 구성 합니다.
